IPv6 對外連線設定(學校內部網路操作)

1.Ubuntu的網路IPv6設定勾選，主機網路 NAT Network

2.進入虛擬主機的proxy設定

cd /etc > cd apt > cd apt.conf.d

3.sudo vi proxy.conf 輸入proxy設定 

1
Acquire::http::Proxy "http://[2001:288:6004:17::53]:3128/"

4.:wq儲存後，sudo apt update 更新apt

5.ifconfig > sudo apt install net-tool ， cd etc > cd netplan 

6.sudo vi 00-installer-config.yaml ，輸入設定

1

network:
  ethernets:
    enp0s3:

        dhcp6: true

                - 2001:b000:168::1
  version: 2

8.sudo netplan apply 更新網路設定，ping6 2001:288:6004:17::2554測試連線

9.VitualBox 新增IPv6連線埠號，連線主機埠號是虛擬主機的IPv6 IP

10.進入cd2020pj1 > git config --global --add http.proxy [2001:288:6004:17::53]:3128

11.git pull 更新，在tmp下 sudo vi oauth_gm.txt 新增檔案

12.sudo vi wsgi.py，近端host改為:: ，修改完後儲存，python3 wsgi.py 開啟連線

13.網址 https//[::1]:18443/along ，即可連到虛擬主機cd2020pj1
